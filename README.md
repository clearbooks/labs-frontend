### Example
```bash
yo angular-typescript:controller login
```

## Tooling

* You need npm (brew install npm)
* npm -g install grunt-cli
* npm -g install bower
* npm -g install yo
* gem install compass
 
## Dependencies

* npm install
* bower install

## Build & development

Run `grunt` for building and `grunt serve` for preview.

## Testing

Running `grunt test` will run the unit tests with karma.
